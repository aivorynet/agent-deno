# Changelog

All notable changes to the AIVory Monitor Deno Agent will be documented in this file.

This project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-02-16

### Added
- Automatic capture of unhandled errors and rejected promises
- Manual exception capture API
- Native Deno WebSocket transport (no external dependencies)
- Configurable sampling rate and capture depth
- Environment variable configuration
- Deno 1.30+ support
